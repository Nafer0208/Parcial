JAVA-1995

Creador:James Gosling

Java es un lenguaje de programación ampliamente utilizado en aplicaciones empresariales debido a su solidez y escalabilidad. Además, Java se utiliza en el desarrollo de aplicaciones móviles, aplicaciones web, juegos, sistemas embebidos y soluciones de inteligencia artificial y aprendizaje automático.

Ejemplo:MySQL es un sistema de gestión de bases de datos relacional de código abierto que utiliza el lenguaje de programación Java para su desarrollo.

JAvaScript-1995

Creador:Brendan Eich

JavaScript se utiliza para crear efectos visuales, validar formularios, crear animaciones, agregar interacciones de usuario, manipular el contenido de la página web, realizar solicitudes asincrónicas al servidor y mucho más..

Ejemplo: React es una biblioteca de JavaScript de código abierto que se utiliza para construir interfaces de usuario interactivas para aplicaciones web de una sola página

C-1972

Creador:Dennis Ritchie

Se utiliza para programar sistemas operativos, dispositivos embebidos, aplicaciones de bajo nivel, software de redes y juegos, así como en la programación de sistemas y software de tiempo real. Es un lenguaje de programación de bajo nivel que permite acceder directamente al hardware de una computadora, lo que lo hace ideal para aplicaciones de alto rendimiento. Es eficiente en términos de memoria y velocidad de ejecución, adecuado para sistemas con recursos limitados..

Ejemplo: Unix es un sistema operativo de propósito general que se desarrolló originalmente en los Laboratorios Bell de AT&T a fines de la década de 1960 y principios de la década de 1970. Está escrito en C

C++-1980

Creador:Bjarne Stroustrup

El lenguaje de programación C++ es una extensión del lenguaje C y se utiliza en una amplia variedad de aplicaciones y sistemas informáticos. Se usa en programación de aplicaciones de escritorio, juegos, sistemas embebidos, software de bases de datos, sistemas operativos, aplicaciones web y dispositivos móviles. Es un lenguaje de programación de alto nivel y orientado a objetos, lo que permite la creación de programas con estructura modular y reutilizable.

Ejemplo:Windows. La mayoría de los componentes de Windows, incluido el kernel, los controladores de dispositivo y las principales aplicaciones como el Explorador de archivos, están escritos en C++.

PHP-1994

Creador:Rasmus Lerdorf

PHP es un lenguaje de programación de código abierto utilizado principalmente para el desarrollo de aplicaciones web y la creación de sitios web dinámicos. Se ejecuta en el servidor, lo que significa que el código se procesa en el servidor antes de enviar la página web al navegador del usuario. Se utiliza normalmente en la creación de sitios web dinámicos, como blogs, foros, redes sociales y sistemas de gestión de contenidos. También se utiliza en la programación de aplicaciones web más complejas, como plataformas de comercio electrónico y sistemas de gestión empresarial. Es fácil de usar y flexible, lo que permite integrarse con otras tecnologías web.

PERL-1987

Craedor:Larry Wall

Perl es un lenguaje de programación de propósito general utilizado en aplicaciones informáticas, como desarrollo web, scripts de automatización de tareas, procesamiento de texto y datos y análisis de registro de servidores. Es popular en programación CGI para la conexión entre el servidor web y otros programas en el servidor, scripts de automatización de tareas y herramientas de línea de comandos. Es conocido por su potencia en el procesamiento de texto y datos, análisis de registros de servidores y procesamiento de archivos CSV, XML y otros formatos de datos. Es flexible y escalable, con una gran cantidad de módulos y bibliotecas disponibles
